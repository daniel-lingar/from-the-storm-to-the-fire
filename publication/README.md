## Publication Rule

Do not publish from the raw `README.md` manuscript directly.

The correct workflow is:

1. Preserve the raw source manuscript.
2. Split the manuscript into clean chapter files.
3. Remove duplicate chapters, draft notes, and editor artifacts.
4. Normalize headings and front/back matter.
5. Run a privacy, legal, and content-safety review.
6. Export a clean PDF from the production workflow repository.
7. Review the PDF on phone, desktop, and tablet before upload.

## Target Final Deliverables

- Final PDF ebook for Payhip / direct sale
- Optional EPUB version for reflowable ebook platforms
- Free sample PDF
- Back-cover copy and storefront description
- Short public description
- Long public description
- Content note and memoir disclaimer
