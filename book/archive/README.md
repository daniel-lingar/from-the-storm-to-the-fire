# Archive Folder

This folder is for material removed from the publication edition but not deleted from the project.

Use this folder for:

- duplicate chapter versions,
- alternate drafts,
- cut scenes,
- removed appendices,
- old front matter,
- old back matter,
- editor notes that should not appear in the final ebook.

## Rule

Do not delete meaningful material unless there is a clear reason.

Move it here instead.

The final ebook should be clean.

The archive can stay messy.
